FREE Hosting Setup Guide - Step by Step

📋 Overview - Kya Karna Hai

Is guide mein hum 3 FREE services use karenge:

  • Supabase - FREE PostgreSQL Database (500MB)
  • GitHub - FREE Code Storage
  • Vercel - FREE Hosting (100GB/month)
💰 Total Cost: ₹0/month - Forever FREE!
1

Supabase Account Banao (Database)

Supabase FREE PostgreSQL database dega.

🔗 Supabase.com Kholo

Steps:

  • Click karo "Start your project"
  • "Continue with GitHub" se login karo
  • Authorize karo
📝 Important: GitHub se login karo, aasan hai!
2

Supabase Project Banao

Ab naya database project banao:

  • Organization: Default select karo ya new banao
  • Project Name: nireeti-nest
  • Database Password: Koi strong password dalo (YAAD RAKHO!)
  • Region: Southeast Asia (Singapore) select karo
  • Click karo "Create new project"
⏳ Wait karo 2-3 minutes - Project setup hota hai
3

Database Connection URLs Copy Karo

Project banne ke baad, yeh URLs copy karo:

  • Left side mein "Settings" (⚙️ gear icon) click karo
  • "Database" click karo
  • Neeche scroll karo "Connection string" section tak

Dono URLs copy karo:

# DATABASE_URL (Connection Pooler - Port 6543)

postgresql://postgres.[PROJECT-REF]:[YOUR-PASSWORD]@aws-0-ap-southeast-1.pooler.supabase.com:6543/postgres?pgbouncer=true&connection_limit=1


# DIRECT_DATABASE_URL (Direct - Port 5432)

postgresql://postgres.[PROJECT-REF]:[YOUR-PASSWORD]@aws-0-ap-southeast-1.pooler.supabase.com:5432/postgres

⚠️ Important:
  • [PROJECT-REF] - Yeh apne aap aa jayega
  • [YOUR-PASSWORD] - Jo password diya tha woh dalo
  • Password mein special characters ho toh URL encode karo
4

GitHub Repository Banao

🔗 GitHub New Repo
  • Repository name: nireeti-nest
  • Description: The Nireeti Nest - Home Decor eCommerce
  • Public select karo (FREE hosting ke liye)
  • Click karo "Create repository"

Repository URL note kar lo: https://github.com/YOUR_USERNAME/nireeti-nest

5

Vercel Account Banao

🔗 Vercel.com Kholo
  • Click karo "Sign Up"
  • "Continue with GitHub" select karo
  • GitHub authorize karo
6

Vercel Pe Project Import Karo

  • Vercel Dashboard pe "Add New..." → "Project" click karo
  • "Import Git Repository" section mein apna repo dhundho: nireeti-nest
  • "Import" click karo
7

Environment Variables Add Karo

Deploy se pehle yeh variables ADD karo:

DATABASE_URL

postgresql://postgres.[REF]:[PASS]@aws-0-ap-southeast-1.pooler.supabase.com:6543/postgres?pgbouncer=true&connection_limit=1


DIRECT_DATABASE_URL

postgresql://postgres.[REF]:[PASS]@aws-0-ap-southeast-1.pooler.supabase.com:5432/postgres


NEXTAUTH_SECRET

nireeti-nest-super-secret-production-key-2024


EMAIL_HOST

smtp.zoho.com


EMAIL_PORT

587


EMAIL_SECURE

false


EMAIL_USER

support@thenireetinest.com


EMAIL_PASSWORD

YOUR_ZOHO_PASSWORD_HERE


EMAIL_FROM

The Nireeti Nest <support@thenireetinest.com>


NEXT_PUBLIC_SITE_URL

https://your-site-name.vercel.app

📝 Steps:
  1. "Environment Variables" section expand karo
  2. Key dalo (e.g., DATABASE_URL)
  3. Value paste karo
  4. "Add" click karo
  5. Sab variables ke liye repeat karo
8

Deploy Karo! 🚀

  • Sab variables add karne ke baad "Deploy" button click karo
  • Wait karo 3-5 minutes
  • "Congratulations!" screen aayegi
🎉 Website Ready!
URL milega: https://nireeti-nest-xyz.vercel.app
9

Admin Account Banao

  • Website kholo: https://your-site.vercel.app
  • Go to: /auth/register
  • Apna account banao
👑 First User = Admin!
Pehla registered user automatically Admin ban jayega!

✅ Checklist

🆘 Agar Problem Aaye

Common Issues:

  • Build Failed: Environment variables check karo
  • Database Error: URLs sahi dalo, password check karo
  • Email Not Working: Zoho password sahi dalo

Vercel dashboard mein "Deployments" → "Logs" check karo errors ke liye.
